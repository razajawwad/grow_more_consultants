# Grow More Consultants Frontend

React, Vite, and Tailwind CSS frontend for the Grow More Consultants website.

## Local Development

```bash
npm install
npm run dev
```

The development server runs at:

```text
http://localhost:5173
```

## Production Build

```bash
npm run build
```

The generated static files are written to `dist/`.

## Environment

Create `.env` from `.env.example` and set the API URL:

```env
VITE_API_URL=http://127.0.0.1:5000/api/v1
```
