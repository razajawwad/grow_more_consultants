# Grow More Consultants

Professional full-stack website project for Grow More Consultants.

## Project Structure

```text
grow_more_consultants/
frontend/   # React + Vite + Tailwind CSS UI
backend/    # Node.js + Express API
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
npm install
npm run dev
```

## Notes

- Frontend uses React, Vite, and Tailwind CSS.
- Backend uses Node.js, Express, Zod validation, CORS, Helmet, and rate limiting.
